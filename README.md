# README

## requirements:
python3
cuda?
podman (can use docker with alias docker="podman")


## env vars
You will need to set your `HF_TOKEN`, or use model that is open by setting `LLMDM_MODEL`

## To install the game, you can run:

```
s/install
```

## To play the game, you can run:
```
s/db_up
llmdm
```

## to reset the db, run
```
s/db_flush
```
